# drools
drools规则引擎

# 应用
财务决策:	贷款发放，征信系统  
库存管理:	及时的供应链路  
票价计算:	航空，传播，火车及其他公共汽车运输  
生产采购系统:	产品原材料采购管理  
风控系统:	风控规则计算  
促销平台系统:	满减，打折，加价购
